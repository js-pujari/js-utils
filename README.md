# What is this ?

You can find js utility functions here.

## Installation

`npm i js-utils`

## How to use ?

`
    import { findAvg } from 'js-utils'

    findAvg([12, 13, 14, 15, 16, 17]);
`

### What all utility functions there ?

*findAvg
*toCamelCase
*getPercentage
*formatPriceINR
*getMobileFormat
*GetFirstElement
